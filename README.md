# 🐝 Cyberbee's Rice

Welcome to my personal Arch Linux configuration, optimized for the **Lenovo LOQ** and a **144Hz** lifestyle. This setup is built for a **CSE student** workflow: programming, cybersecurity, and gaming.

## 🖥️ System Specs
- **OS:** Arch Linux (EndeavourOS)
- **WM:** Hyprland
- **Machine:** Lenovo LOQ
- **GPU:** NVIDIA RTX

## 🚀 One-Line Installation
If you want to replicate this setup, clone the repo and run the installer:


git clone [https://github.com/Cyberbee-pro/dotfiles.git](https://github.com/Cyberbee-pro/dotfiles.git) 
./install.sh
